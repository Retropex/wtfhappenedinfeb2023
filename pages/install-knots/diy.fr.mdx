## instruction noeud DIY

Si vous avez fait votre propore noeud sur linux voici un [guide](https://jesterhodl.com/raspibolt-replacing-bitcoin-core-with-knots/) simple et rapide pour passer de Bitcoin Core à Bitcoin Knots.

Si vous voulez créer un noeud Bitcoin depuis zéro sur alpine Linux vous pouvez utiliser ce [guide](https://microbolt.guide/en-US), mais assurez vous d'utiliser [Knots](https://microbolt.guide/en-US/bitcoin/bitcoin-client/bitcoin-knots) ou d'appliquer le [patch ordisrespector](https://microbolt.guide/en-US/bitcoin/bitcoin-client/bitcoin-core#apply-patches) sur Core.

Il y a aussi un guide pour Ubuntu/Debian qui est un peu plus facile qu'alpine linux. Vous pourrez trouver ce guide [ici](https://scratch-knots.orangepill.ovh).
