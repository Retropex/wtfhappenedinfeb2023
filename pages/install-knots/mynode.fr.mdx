## Instruction pour mynode

Mynode ne supporte pas encore Knots, donc en attendeant vous pouvez utiliser ordisrespector pour limiter le spam.

Vous devez avoir au minimum la version [v0.3.10](https://github.com/mynodebtc/mynode/blob/776ccb35d1939b76de429dd129247425cd911f17/CHANGELOG#L4)

1. Connectez vous à mynode
2. Allez dans les réglages
3. Cliquer sur `Show upgrade logs`
4. Allez sur custom bitcoin version
5. Selectionner la version d'ordisrespector désiré 
6. Cliquez sur install

Et voilà, vous avez maintenant ordisrespector sur votre noeud.
