## Instruction pour start9

Sur votre serveur start9:

1. Télécharger le packet ([.s9pk](https://github.com/Retropex/ordisrespector-startos/releases/tag/v26.1.knots20240325))
2. Allez sur system -> sideload a service -> browse
3. Selectionnez le fichier .s9pk
4. CLiquez sur upload & install
5. cliquez sur Bitcoin Knots -> configure -> save -> start

Et voilà, vous avez Bitcoin Knots.
